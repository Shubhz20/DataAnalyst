## How to populate commit/PR counts

1. Go to the repo → Insights → Contributors tab for commits
2. Pull Requests tab → filter by `author:<username>` for PRs opened
3. Pull Requests tab → filter by `reviewed-by:<username>` for PRs reviewed

## Integrity note

This matrix must match GitHub's actual history. If a teammate's commit count shows as zero on GitHub Insights, writing "high contribution" here will trigger a deduction per the project rules.
